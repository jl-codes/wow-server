How to Join

Four steps. That's all it takes.

01

Download the Client

Get the WoW 3.3.5a client (~15 GB). Not the modern retail version.

ChromieCraft Download  ·  Warmane Download

If the download is 100GB+, it's the wrong version.

02

Set the Realmlist

Open Data/enUS/realmlist.wtf in your WoW folder. Replace everything with:

set realmlist 127.0.0.1
03

Get an Account

Contact the server admin, or create one via the worldserver console:

account create <username> <password>
04

Launch & Play

Run Wow.exe — not the Launcher. Log in. Create a character. Adventure awaits.
